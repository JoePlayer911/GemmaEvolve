# Patch Notes - 2026-01-17

## New Features

### Verilog Optimization Support
Added a complete Verilog code evolution example using Icarus Verilog (`iverilog`) for functional verification.

**New Files:**
- `examples/verilog_optimization/gemma_config.yaml` - Configuration for Verilog evolution
- `examples/verilog_optimization/initial_program.v` - Initial dummy `popcount16` module
- `examples/verilog_optimization/testbench.v` - 100-test functional verification testbench
- `examples/verilog_optimization/evaluator.py` - Python wrapper calling `iverilog` and `vvp`
- `run_gemma_verilog.sh` - Convenience script for running Verilog optimization

**Key Features:**
- Full Verilog-2001 compliance
- Automatic compilation and simulation via Icarus Verilog
- Accuracy-based scoring with conciseness bonus
- Explicit syntax rules in system prompt to prevent common LLM errors

**Usage:**
```bash
./run_gemma_verilog.sh          # Default 50 iterations
./run_gemma_verilog.sh 100      # Custom iteration count
```

**Requires:** `sudo apt install iverilog`
